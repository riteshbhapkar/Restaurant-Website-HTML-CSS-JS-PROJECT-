# Restaurant Website Project

This project is a responsive website for a restaurant, built using HTML, CSS, and JavaScript. The site provides essential information about the restaurant, ensuring a user-friendly experience across various devices. 

## Live Demo
Check out the live site [here](https://riteshbhapkar.github.io/Restaurant-Website-HTML-CSS-JS-PROJECT-/).

## Features

- **Responsive Design**: The website adjusts seamlessly to different screen sizes, providing a great user experience on both mobile and desktop devices.
- **Smooth Navigation**: A sticky navigation bar allows easy access to all sections of the website, improving the user experience.
- **Home Section**: Introduces the restaurant with a welcoming message and high-quality images.

- **Awards Section**: Highlights any accolades or recognitions the restaurant has received. (Currently blank)
- **Contact Section**: Offers the restaurant's contact details, including address, phone number which is clickable.

## Technologies Used

- **HTML**: For the structure of the website.
- **CSS**: For styling the website and ensuring a responsive design.
- **JavaScript**: For enhancing interactivity and smooth navigation.
